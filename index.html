HAWK

Photo: Iftekhar Naim

HAWK (hitting associations with k-mers) is an alignment free method for association mapping from sequencing reads.

Source Download

Binary(Linux) Download

HAWK is described in the paper
Atif Rahman and Lior Pachter, Association Mapping From Sequencing Reads Using K-mers, http://www.biorxiv.org/content/early/2017/05/23/141267.

Git repository of HAWK is at https://github.com/atifrahman/HAWK.
